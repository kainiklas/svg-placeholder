    SVG Placeholder 
    The customizable & colorful placeholder

    You are building a website, mockup or prototype? Give it a colorful touch with SVG placeholder.

    Choose among 4 styles and customize the color to your needs. You can also set a pre-defined width and height of the placeholder image.

    Example usage: <img src="https://svg-placeholder.web.app/angular">

    Embed SVG Placeholder

    Use the following URL to embed the SVG placeholder on your page:

    • https://svg-placeholder.web.app/{style}/{preset}?width={width}&height={height}&c1={c1}&c2={c2}

    The parameters have the following meaning. Query parameters are optional. If not provided, the default is used:

    • {style} = [angular, burst, linear, stripes]
    • {preset} = [1, 2, 3, 4] (optional)
    • {width} = [number] (optional) (e.g., 300)
    • {height} = [number] (optional) (e.g., 300)
    • {c1} = [hex-html-color] (optional) (e.g., #123456)
    • {c2} = [hex-html-color] (optional) (e.g., #123456)
    • {deg} = [0-360] (optional) (degree of the linear gradient from c1 to c2)
